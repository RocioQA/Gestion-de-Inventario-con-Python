# Inventario de Productos

Aplicación creada con Python para Gestión de Inventario con base SQLite.

# Instrucciones

1. Instalar dependencias:
```bash
pip install colorama
```

2. Ejecutar la aplicación:

- Opción 1 por consola:
```bash
python inventario.py
```

- Opción 2 por archivo ejecutable desde la carpeta del código

## Funcionalidades

- Registrar producto
- Ver todos los productos
- Buscar producto por ID
- Buscar por nombre o categoría
- Actualizar producto
- Eliminar producto
- Reporte de stock bajo

La aplicación crea el archivo `inventario.db` automáticamente en el mismo directorio.
